Cara kerja sistem kode:
Kode baru yang kamu generate akan disimpan di localStorage browser ini sebagai "extra codes". Peserta yang membuka file tryout di browser yang sama dengan browser ini bisa menggunakan kode baru tersebut. Untuk distribusi antar komputer, copy kode baru dan tambahkan manual ke array VALID_CODES di file toefl_unpad_jwoobim.html.

Instruksi Tambah Kode ke File Utama
Langkah:
1. Buka file toefl_unpad_jwoobim.html dengan text editor (Notepad, VSCode, dll.)
2. Cari teks: const VALID_CODES = [
3. Tambahkan kode baru di dalam array tersebut, pisahkan dengan koma
4. Simpan file → kode baru langsung aktif

Contoh format kode yang valid: JW-XXXX-XXX (panjang 12 karakter dengan 2 tanda hubung)
Perhatian: Operasi reset bersifat permanen dan tidak dapat dibatalkan. Gunakan hanya jika benar-benar diperlukan.
🔄 Reset Satu Kode
Hapus data perangkat yang terdaftar untuk satu kode tertentu. Kode tersebut bisa digunakan kembali di perangkat baru.
🗑️ Reset Semua Data
Hapus SEMUA data penggunaan kode di browser ini. Semua kode akan kembali ke status "belum dipakai". Data peserta dari browser lain tidak terpengaruh.
➕ Daftarkan Perangkat Manual
Tambahkan Device ID tertentu ke kode tertentu. Berguna jika peserta ganti perangkat.
